## Decisions prises

### 1. Migration NextAuth -> Better Auth
- NextAuth v4 est legacy, migration prevue vers Better Auth
- Impact : src/lib/auth.ts, route API auth, client, middleware
- Adapter Prisma : `betterAuth({ database: prismaAdapter(prisma, { provider: "sqlite" }) })`

### 2. Base de donnees prod : Turso
- SQLite ne marche pas sur Vercel (filesystem read-only)
- Turso = libSQL (fork SQLite), migration minimale
- Config conditionnelle : SQLite local en dev, Turso en prod

### 3. PDF sur Vercel : OK
- @react-pdf/renderer fonctionne sur serverless Vercel
- Fonts via URL (pas filesystem), timeout 10s suffisant

### 4. Deploiement : Vercel via GitHub
- Variables d'env : TURSO_DATABASE_URL, TURSO_AUTH_TOKEN, BETTER_AUTH_SECRET, BETTER_AUTH_URL

### 5. UI : Composants natifs Next.js
- Multi-selection de composants dans le formulaire de devis

## Blocage identifie
